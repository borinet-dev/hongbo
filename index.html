아래 이미지 클릭시 홈페이지로 이동합니다!

첫번째 방법:

1. 우선 맨 위로 이동하신 뒤 ========= 점선을 기준으로하여
아래쪽을 그림과 같이 선택 및 복사(컨트롤+C 또는 우클릭 복사)를 합니다.



2. 홍보사이트로 이동하여 [리니지2 홍보 게시판] 등으로 이동 후 글작성을 누릅니다.

3. 제목을 작성 후 본문에서 컨트롤+V 를 이용하여 붙여넣기 합니다.




두번째 방법:

1. 첫번째 방법처럼 붙여넣기가 안되는 사이트가 있습니다.

   이럴경우에는 홍보이미지를 첨부하여 글을 작성할 수 있습니다.

2. 다운로드.

위 파일은 732KB 용량의 이미지 파일입니다.


3. 홍보사이트로 이동한 후 글쓰기 버튼을 눌러서 해당 파일을 첨부합시다.

  - 본문 아래쪽에 [파일 선택] 또는 [파일 추가] 버튼을 눌러서 위에서 내려받은 파일을 첨부하면 됩니다.



세번째 방법:

1. html 의 기능을 이용한 글 작성방법입니다.

  - https://freebox4.xyz/ 사이트의 경우가 해당됨.

2. 글 작성으로 이동하신 후 html 에 체크를 합니다.

3. 제목을 입력 후 본문에는 아래의 코드를 복사해서 붙여넣어 줍니다.
